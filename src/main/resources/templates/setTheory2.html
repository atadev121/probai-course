2.3.12 Cardinality of a set

The cardinality of a set is the number of elements in this set, it's denoted by |A|, for finite sets we have:
\( 1.☛ |A_{1} \times A_{2} \times .. \times A_{n}|=|A_{1}|*|A_{2}| *..*| A_{n}| \)
\( 2.☛ |A \cup B|=|A|+|B|-| A \cap B| \) 📘

2.4 Functions

2.4.1 What is a function

A function is a binary relation 📘 a binary relation over sets X and Y is a set of ordered pairs (x,y) where\( \;x \in X\;and\; y\in Y \)
(x,y) is an ordered pair means that \( x\neq y \Rightarrow (x,y)\neq (y,x)\) or association between two sets X and Y which associates every element in set X to exactly one element in set Y.
\( 1.☛ \;f:A \rightarrow B, \;\; A:Domain\;,B:Co-domain, \) the Range is all possible values of f(x) with x ∈A. 📘 And we have \(R_f=\left\{ y \in B \setminus \exists x \in A \;such \;as \; f(x)=y \right\}\) (the range is a subset of B).
2.☛ Injective function: f is an injection if for every element y in \( R_f\) there is only one element x in A such as f(x)=y.
3.☛ Surjective function: f is a surjection if for every element y in B there is at least one element x in A such as f(x)=y. 📘 It's the same as writing \(R_f =B\)
4.☛ Bijective function:We say that f is a bijection if it is both injective and surjective.

2.4.2 Cardinality revisited

1. Two sets are said to be equi-cardinal (|A|=|B|) if there is a bijection from A to B.
2. B has a greater cardinality than A \((|B| \geqslant |A|) \) if there is an injection from A to B.
3. B has a strictly greater cardinality than A (|B| > |A|) if there is an injection from A to B but there is no surjection from A to B. 📘 A and B might not be finite

2.5 Combinatorics

2.5.1 Introduction

We are going to see further in this course that if S is a finite sample space with equally likely outcomes, then for any event A we have: \(P(A)={|A| \over |S|} \). So calculating probabilities consists simply of counting number of possibilities.
For example a random experiment consists of drawing k balls from a box containing n balls. This experiment differs depending on the way we draw the balls, and how we make distinction between outcomes.

2.5.2 Ordered sampling without replacement

Ordered sampling without replacement📘 If the balls are numbered, and order makes a distinction between outcomes, that is (b1,b4,b5) is different from (b4,b1,b5) ( k=3, n=5) and when we draw a ball we don't put it back in the box before making the following draw (suppose we draw b1 in the first draw, we don't put it back in the box before doing the second draw), an example of this sampling method could be the following:
20 people are racing, the first, second and third in this race will have different prizes depending on their rank, how many possibilities are there? is called a permutation and the number of possibilities in this sampling method is given by the formulae: \( P_{k}^{n}={n! \over (n-k)!} \), Proof:📘 we have n possibility for the first draw, (n-1) for the second, and (n-(k-1)) for the kth draw so there are n.(n-1).(n-2)..(n-(k-1)) possibilities, that is \( {{1.2.3..n} \over {1.2..(n-k)}} ={ {n!} \over {(n-k)!}} \)

2.5.3 Unordered sampling without replacement

Unordered sampling without replacement 📘 If the balls are numbered, but order makes no distinction between outcomes, that is all we care about is if a ball is drawn or not, that is (b1,b4,b5) is the same as (b4,b1,b5) (we suppose k=3 and n=5) and when we draw a ball we don't return it in the box before making the following drawing, ( suppose we draw b1 in the first drawing, we don't put it back in the box before doing the second draw), an example of this sampling method could be the following:
20 people are racing, the first three people will have a prize (that is the first, the second and the third will have identical prizes), how many possibilities are there ? is called a combination and the number of possibilities in this sampling method is given by the formula: \( \binom{n}{k}={n! \over k!(n-k)!} \).
Proof and examples:📘
In this case order is not a distinguishing factor between possible outcomes, that is if n=5 and k=3, (b1,b3,b5) is the same as (b3,b5,b1) and each outcome corresponds to \( P_{k}^{k}=k! \) permutations so \( \binom{n}{k}.k!={n! \over (n-k)!} \)
Figure 12 gives all possible outcomes in the case of four balls and three draws with order and without replacement, in this case the number of all possible outcomes is 24 📘 \(\frac{4!}{(4-3)!}= \frac{4!}{1!}=\frac{1*2*3*4}{1}=24\) .
To get the possibilities of unordered draws we must get rid of redundant outcomes for example (b1,b3,b4) and (b4,b3,b1) and (b3,b1,b4).

2.5.3.1 Number of distinct sequences

The number of distinct sequences formed by n entities A and p entities B is \( \binom{n+p}{p}\).
Proof and example:📘
Distinct sequences imply just order in a given number of two or more different entities, for example, we have two different entities A and B, we have 4 A's and 6 B's, we want to count the number of all different sequences, for example:
AABABBABBB, BBBAABABAB, BABABABABB,..It's like putting the A's and B's in ten ordered cases (or digits if A and B were numbers, for example 1151551555,..).
A way to form a sequence is to place all A's in four cases, then to put the B's in the cases left, that is to choose 4 cases from the 10 cases where we will put the A's, then put the B's in cases left, choosing 4 cases from 10 cases correspond to: \( \binom{10}{4}\) outcomes.

2.5.4 Ordered sampling with replacement

The number of possibilities in this sampling method is given by the formula:\( n^k \), Proof:📘 In each draw (k draws) we have n possibility
Example: 📘 In a presidential debate, 20 journalists are preparing to ask three questions to the candidates, a journalist can ask more than one question.
How many possibilities are there? The first question can be asked by any of the 20 journalist, so there is 20 possibilities, the second one can also be asked by any one of the journalist so there is 20 possibilities, the same for the third, so the total number of possibilities is 20*20*20, hence the general formulae \( n^k \).
If this experiment were designed with no replacement, a journalist can ask only one question
If this experiment were designed without order,that means, there is no difference as what question was asked by which journalist, that is journalist A asks question 1, B asks question 2 and C asks question 3 (A1, B2,C3) :, is the same as (A3,B1,C2) 📘Caution! You must be careful as to how the design of the experiment is done, as the outcomes can differ significantly according to the design, For example: suppose that in this same situation, we pick a journalist randomly (20 possibilities) and we invite him to ask the three questions in the order he sees fit \( P_3^3 =6\) possibilities, so there is 60 possibilities
As an application of this formula, the number of subsets in a set of n elements is \( 2^n \) 📘 While choosing a subset, for every element in this subset there is two possibilities, either this elements is included or not in this subset, so there is \( 2^n \) possibility
