2 Set theory: Review

2.1 What is a set

A set is a collection of some elements for example: Set1={↫,↭,♣,♠,⛅,123,AED,John} ,Set2={\(x^2\), where x∈ℝ}📘 ,so Set2 is the set of positive real numbers (ℝ is the set of real numbers) \(Set3=\left \{ x^3, where \;x ∈(1,3,5)\right \} \) 📘 , so Set3={1,9,512}

2.2 Special sets

2.2.1 Universal set

The set of all possibly imagined elements 📘 that we can consider in a given context is called the universal set, it's denoted by Ω or S

2.2.2 Empty set

The null set is a set with no elements, it's denoted by \(\phi \;or \left\{ \right\} \) , for any set A, we have \(\phi \subset A\)

2.2.3 Distinct sets

Two sets are distinct if they are different, that means there is at least one element that belongs to one of the two sets and doesn't belong to other.📘 not to mix with disjoint, see later.

2.2.4 Disjoint sets

A sequence of sets \( A_{1}, A_{2},A_{3},.. \) are considered to be disjoint if \( A_{i} \cap A_{j}=\varnothing, \vee i\neq j \) 📘 See mutually exclusive events for comparison of the two concepts.

2.3.2 Inclusion

We say set A is a subset of set B if all elements in set A belong to set B, we write:\(A\subset B\) 📘 A is included in B, with mathematical expressions we write:\((\vee x\in\Omega, x\in A\Rightarrow x\in B)\Leftrightarrow (A\subset B\))
Example: A={1,2,jhk,↫,↭}, B={2,jhk,↭}, we have \(B\subset A\)
* For two subsets A and B we have: \( A=B\;\Leftrightarrow \;A \subset B \) and \(B \subset A\) 📘 because\((A= B) \Leftrightarrow (\vee x\in \Omega, x\in A \Leftrightarrow x\in B \Leftrightarrow A \subset B \;and\; B \subset A ) \)

2.3.3 Union

The union of two sets A and B is a set consisting of all elements in set A and all elements in set B
The union is denoted by \(A\cup B\), and we have \( \vee x\in \Omega , x\in A\cup B\Leftrightarrow x\in A \;or\; x\in B \)
For example if A={1,"R","ED"} and B={3,4,"T"}, \( A\cup B=\left\{1,"R","ED",3,4,"T" \right\} \)

2.3.4 Intersection

Intersection of two sets A and B is a set consisting of all elements that belong to both sets, this set is denoted by \( A \cap B\).

2.3.5 Subtraction (Difference)

The subtraction of a set B from a set A is a set consisting of all elements that belong to set A but not to set B, it's denoted by A-B.

2.3.6 Complement

The complement of a set A is a set consisting of all elements but not those belonging to A, it's denoted by \( \overline{A} \;or\; A^{c} \)
We have \( A-B=A\cap \overline{B} \)

2.3.7 Mutually exclusive

The terms Mutually exclusive and disjoint are used interchangeably.

2.3.8 Partition

A partition is a group of sets that satisfy the following conditions:
-They are disjoint
-Their union is equal to the universal set

2.3.9 Morgan's Law

For any n sets we have:
\(*\;\overline{A_{1}\cup A_{2}\;\cup ..A_{n}}=\overline{A_{1}}\cap \overline{A_{2}}..\cap\; \overline{A_{n}} \\ * \overline{A_{1}\cap A_{2}\;\cap ..A_{n}}=\overline{A_{1}}\cup \overline{A_{2}}..\cup\; \overline{A_{n}} \) proof: 📘 \(x\in \overline{A\cup B}\Leftrightarrow x\notin (A\cup B)\Leftrightarrow \neg [x\in (A\cup B) ]\Leftrightarrow \neg[x\in A \;or\; x\in B]\) \( \Leftrightarrow x\notin A \;and\; x\notin B \Leftrightarrow x\in \overline{A}\;and\; x\in \overline{B} \) \(\Leftrightarrow x\in (\overline{A}\cap \overline{B}) \) ( \( \neg \) is the negation operator.)
we prove the general case by recurrence
Example:📘 \( \text{If A, B and C are three sets where }A=\left\{1,3,5,67,8,9,32 \right\},\) \( B=\left\{5,7,22 \right\} \;and\;C=\left\{90,34,8 \right\} \) and \(S= \left\{1,3,5,67,8,9,32,7,22,90,34\right\}\) \(then: \\ \quad *\overline{A}= \left\{7,22,90,34\right\} \\ \quad *\overline{B}= \left\{1,3,67,8,9,32,34,90\right\} \\ \quad *A \cup B=\left\{1,3,5,67,8,9,32,7,22\right\} \\ \quad *A \cup B \cup C=S \\ \quad *A \cap B=\left\{5\right\} \\ \quad *\overline{A} \cap \overline{B}=\left\{34,90\right\}=\overline{A \cup B} \\ \quad *\left\{1,5,7,3,67\right\}, \left\{22,8\right\}, \left\{8,9,32,1\right\} are\; disjoint \)

2.3.10 Laws related to union and intersection of sets

2.3.10.1 Distributive law

a) \( A \cup (B \cap C)=(A \cup B) \cap (A \cup C) \)
b) \( A \cap (B \cup C)=(A \cap B) \cup (A \cap C) \) 📘

- \( blue=B \cup C\) (figure 3)📘 The blue part in figure 3 is the union of B and C
- \( red=A \cap (B \cup C)\) (figure 4)📘 The red part in figure 4 is obviously the intersection of A and the union of B and C
- \( yellow=A \cap C\) (figure 2)📘 The yellow part in figure 2 is the intersection of a and C
- \( pink=A \cap B\) (figure 1)📘 The Pink part in figure 1 is the intersection of A and B
- \( red=pink \cup yellow \)
c) If \( B_{1},B_{2},.. \) is a partition, then for any event A we have: \( A=(A\cap B_{1}) \cup (A\cap B_{2}) \;\cup ... \)📘 Every element in A belongs to one of the sets of the partition.

2.3.10.2 Commutative law

a) \( A \cup B = B \cup A \)
b) \( A \cap B = B \cap A \)

2.3.10.3 Associative law

a) \( (A \cup B) \cup C = A \cup (B \cup C) \)
b) \( (A \cap B) \cap C = A \cap (B \cap C) \)

2.3.11 Cartesian Product

The Cartesian product of any n sets in some given order is the union of all n-tuple formed by all elements in those n sets in that specified order: The Cartesian product of sets \(A_{1}, A_{2},..A_{n}\) in this order is denoted by \(A_{1} \times A_{2} \times .. \times A_{n}\) and is defined as the following: \(A_{1} \times A_{2} \times .. \times A_{n} =\left\{ (a_{1},a_{2},..,a_{n}),where\; a_{i} \in A_{i}\right\} \)
